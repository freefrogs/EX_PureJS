## :books: PureJS task

Simple router-based aplication, consisting of two views (home & welcome).

### Used technologies:
* PureJS with Webpack,
* JavaScript (ES5+),
* CSS, HTML.

### Install & usage (bash)
1. Clone repository
```
git clone git@github.com:freefrogs/EX_PureJS.git
```
2. Install required project dependencies
```
cd EX_PureJS
npm install
```
3. Run developer server
```
npm run dev
```
